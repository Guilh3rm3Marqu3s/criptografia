O repositório consiste em uma implementação de algoritmos de criptografia estudados na disciplina de Segurança da Informação. A aplicação é baseada em Python e utiliza o framework `CustomTkinter` para a implementação de interfaces gráficas.

# Como Usar

## Instalação

Baixe os arquivos da branch `main` desse repositório. Em seguida navegue para o diretório raiz do projeto e instale as depend6encias necessárias através do seguinte comando:

```bash
pip install -r requirements.txt
```

Após o término da instalação das dependências, execute o seguinte comando (na raiz do proejto):

```bash
python3 main.py
```
